CSS盒模型是什么

1、包含content、padding、border、margin几个要素
2、分为标准盒模型和IE盒模型

标准盒模型和IE盒模型的区别

1、标准盒模型：宽、高只是content的宽、高
2、IE盒模型：宽/高 = content+padding+border

CSS如何设置两种类型

1、标准盒模型：box-sizing：content-box；
2、IE盒模型： box-sizing：border-box；
