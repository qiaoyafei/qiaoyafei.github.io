DOM事件

事件处理程序： 事件就是用户或者浏览器自身执行的某种动作。如click、load等都是事件的名字。而相应某个事件的函数就叫做事件处理程序

DOM事件级别

DOM0 HTML标签上添加click事件
element.onclick = function(){}

DOM2 事件处理
element.addEventListener(‘click’,function(){},false)
element.removeEventListener(‘click’,function(){},true)

DOM3 element.addEventListener(‘keyup’,function(){},true)

DOM事件模型

捕获： window ——> document ——> html ——> body ——> 目标元素
冒泡： 目标元素 ——> body ——> html ——> document ——> window

DOM事件流

捕获 ——> 目标阶段 ——> 冒泡

Event对象常见应用

event.preventDefault()  //阻止默认事件
event.stopPropagation() //阻止冒泡行为
event.stoplmmediatePropagation()
