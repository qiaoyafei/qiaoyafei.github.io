渲染机制

  |  
 阅读次数

什么是DOCTYPE及它的作用

DTD 告诉浏览器是什么文档类型
DOCTYPE告诉浏览器当前文档用使用什么样的文档类型定义（DTD）

HTML5 的写法 <!DOCTYPE html>
HTML4 分为严格模式、传统模式
严格模式：包含html元素和属性，不包含展示性和弃用的元素
传统模式：包含html元素和属性，包含展示性和弃用的元素

浏览器的渲染过程

重排Reflow

DOM结构中的各个元素都有自己的盒子，这些都需要浏览器根据各种样式来计算并根据结果将元素放到该放的位置，这个过程称为Reflow

Reflow出发条件

1、当增加、删除、修改DOM节点时，会触发Reflow 或 Repaint
2、当移动DOM位置，或者搞个动画
3、修改css样式时
4、Resize窗口的时候（移动端没有这个问题），或者滚动时
5、修改网页的默认字体

Repaint

页面要呈现的内容，通通画在屏幕上

触发Repaint

1、DOM改动
2、CSS改动

如何最小Repaint

做修改时，一次性添加片段，尽量减少改动次数
